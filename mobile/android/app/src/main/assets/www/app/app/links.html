Реальная причина, почему не получается войти/зарегистрироваться: Сайт на GitHub Pages — это только HTML/JS (статика). Регистрация и вход идут через API (бэкенд в папке backend-go). Этот бэкенд нигде не развёрнут в интернете — он работает только у тебя на компе по команде go run .. Поэтому: либо открывай сайт с того же компьютера, где запущен бэкенд, и вводи http://localhost:3000; либо задеплой backend-go на Render/Railway и вставь выданный URL — тогда вход будет работать с любого устройства.

Бэкенд: в папке backend-go выполни go run ., потом открой «Вход» или «Тест» с этого же компьютера.
