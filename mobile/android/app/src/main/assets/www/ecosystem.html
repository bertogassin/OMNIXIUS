What’s live now

Mail, Marketplace, Shop (orders), Profile — same app, same API. More directions are added over time.

  • Mail — Internal conversations and messages. → Mail
  • Marketplace — Listings, search, categories. → Marketplace
  • Shop — Buy and sell, orders, “Contact seller” (opens Mail). → Orders
  • Profile — Account, avatar, my orders. → Profile

How new directions connect

Additional projects and verticals attach to one or more platforms so the whole product stays unified.

  • Connect — Social, learning, media, community, support.
  • Trade & Finance — Payments, investments, rewards, subscriptions.
  • Repositorium — Files, storage, compute, backups.
  • Blockchain (IXI) — Tokens, governance, startups, staking.

Cross-cutting examples: Education (Connect + Trade + Repositorium), Startups (Blockchain + Trade + Connect), Media (Connect + Trade + Repositorium). One login for all.
