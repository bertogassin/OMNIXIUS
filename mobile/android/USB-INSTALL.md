# Установка на телефон по USB

## Вариант 1: Через Android Studio (рекомендуется)

1. Подключи телефон по USB.
2. На телефоне: **Настройки → Для разработчиков → Отладка по USB** (включи).
3. Открой **Android Studio** → **File → Open** → папка `SITE OMNIXIUS\mobile\android`.
4. Дождись синхронизации Gradle.
5. Вверху выбери своё устройство и нажми **Run** (зелёный треугольник).  
   Приложение соберётся и установится на телефон.

## Вариант 2: Сайт в браузере (без установки)

1. Компьютер и телефон в одной Wi‑Fi.
2. На компе запусти бэкенд и фронт (`start-app.bat` или вручную).
3. На телефоне в браузере открой: **http://192.168.1.10:5173/app/**  
   (подставь IP своего ПК из `ipconfig`).
4. Если просят Backend URL: **http://192.168.1.10:3000**.

---

Если в консоли `gradlew installDebug` выдаёт ошибку про JAVA_HOME или jlink — используй Android Studio (вариант 1).
