By purpose

Architecture

Four platforms and technology stack. Details only.

1. OMNIXIUS CONNECT

Social network + expert platform.

  • Expert profiles, real-time client matching, orders and reviews. AI matching by skills, rating, location.
  • “Waiting for orders” + geolocation — find specialists nearby. Ratings and reviews.
  • Chat, audio and video calls. Consultation recording, smart contracts for deals.
  • Marketplace: goods, courses, NFT, services. Crypto, FIAT, cards, PayPal. Subscriptions and exclusive content.
  • Security: smart contracts, AI moderation.

2. OMNIXIUS TRADE & FINANCE

Exchange, investments, copy trading.

  • Trading: crypto, gold, oil, stocks, indices. Startups. Buyback fund for IXI blockchain economy.
  • Copy trading, long-term portfolios, 1% income auto-investing with AI.
  • Wallets (crypto + fiat), Apple Pay, Google Pay, PayPal, cards.

3. OMNIXIUS REPOSITORIUM

Decentralized cloud and server network.

  • Server devices at users’ homes. Rewards for storage and compute. Global cloud without centralized data centers.
  • IPFS + AI encryption. Blockchain access control. Data recovery.
  • Compute rental for companies and startups. Referral programs.

4. OMNIXIUS BLOCKCHAIN (IXI)

IXI blockchain — base platform, not a token.

  • Zero-Knowledge Proofs (ZKP) — privacy.
  • Hybrid Proof-of-Stake + Proof-of-Storage — earn for storing data.
  • AI transaction monitoring.
  • 30-year emission, buyback fund for economic stability.

Technology stack (only)

SPARK, RUST, C++, SWIFT, GO. No other languages in the codebase.

  • Backend / API / Services: Go, Rust, C++.
  • Data / Analytics: Spark.
  • Mobile (Apple): Swift (iOS).
  • Blockchain / low-level: Rust, C++.

Site and APIs support all languages; English is the fallback.
