# OMNIXIUS — Web (React + TypeScript)

Заготовка веб-клиента в браузере: Vite + React + TypeScript. Текущее приложение — `app/` (HTML/JS); здесь — основа под будущую миграцию или второй клиент.

**Уровень:** начало; один экран. Стек по направлениям: ARCHITECTURE.md.

```bash
npm install
npm run dev
# http://localhost:3000 (Vite default)
```
