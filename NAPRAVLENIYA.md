# OMNIXIUS — все направления (как было с самого начала)

Полная карта: 4 платформы, блокчейн IXI, игры, обучение, стартапы, медиа, награды, стек. Один аккаунт, одна экосистема. Источники: ARCHITECTURE.md, ECOSYSTEM.md, ROADMAP.md, OMNIXIUS_CHECKLIST.md.

---

## 1. Четыре платформы (ядро экосистемы)

### OMNIXIUS CONNECT
Люди и сервисы: соцсеть + экспертная платформа.

| Что входит | Детали |
|------------|--------|
| Профили, гео | Заказы по геолокации, рейтинги, отзывы. |
| Чат, аудио/видео | Консультации, запись, смарт-контракты. |
| Маркетплейс | Товары и услуги: crypto, FIAT, карты, PayPal, подписки. |
| Безопасность | Смарт-контракты, модерация через ИИ. |

**В репо сейчас:** Mail, Marketplace, Shop, Profile (`app/`, `backend-go/`). Дорабатывать: geo, видео, подписки.

---

### OMNIXIUS TRADE & FINANCE
Деньги и активы: биржа, кошельки, инвестиции.

| Что входит | Детали |
|------------|--------|
| Биржа | Крипто, золото, нефть, акции, индексы, стартапы. |
| Buyback fund | Экономика блокчейна IXI. |
| Copy trading | Долгосрочные портфели, авто-инвестиции с ИИ. |
| Платежи | Кошельки, Apple Pay, Google Pay, PayPal, карты. |

**В репо сейчас:** заготовки в видении; API заказов есть. Дальше: кошельки, биржа (MVP), copy trading.

---

### OMNIXIUS REPOSITORIUM
Данные и инфраструктура: децентрализованное хранилище и вычисления.

| Что входит | Детали |
|------------|--------|
| Домашние серверы | Награды за хранение и compute. |
| IPFS + шифрование ИИ | Доступ через блокчейн. |
| Аренда compute | Реферальные программы. |

**В репо сейчас:** только в ROADMAP/ECOSYSTEM. Дальше: узлы хранения, награды, аренда compute (MVP). Стек: Rust/C++, Spark для аналитики.

---

### OMNIXIUS BLOCKCHAIN (IXI)
**IXI — это блокчейн (базовая платформа), не отдельный токен.** Основа доверия и экономики.

| Что входит | Детали |
|------------|--------|
| Zero-Knowledge Proofs (ZKP) | Приватность и верификация. |
| Консенсус | Hybrid Proof-of-Stake + Proof-of-Storage. |
| Мониторинг ИИ | Над транзакциями. |
| Эмиссия и buyback | 30 лет эмиссии, buyback fund. |

**В репо сейчас:** в видении и ROADMAP; Phase 1 — прототип IXI (Rust/C++): consensus, ZKP, экономика. В backend-go уже есть PQC (Dilithium3) для токенов — квантово-устойчивая криптография.

## 6. Стек (приоритеты)

- **Приоритет 1 (ядро):** Go (API), Rust (движки), Spark (аналитика).
- **Приоритет 2 (клиенты):** TS/JS + React/Vue (`app/`, `web/`), Swift (iOS), Kotlin (Android).
- **Приоритет 3 (инфра + ИИ):** Python, Bash, YAML; Docker, K8s, Terraform, CI/CD.
- **Игры (если будут):** C++ (Unreal), C# (Unity).
- **Квантовая устойчивость:** PQC (Dilithium3), Argon2id, см. QUANTUM_READINESS.md.

## 7. С чего начать прямо сейчас

1. **Сайт** — страницы, навигация, мобилка.  
2. **Маркетплейс и заказы** — объявления, заказы, связь с почтой.  
3. **Почта** — диалоги, уведомления.  
4. **Профиль** — восстановление пароля, настройки.  
5. **ИИ** — стабильный чат.  
6. **Деплой бэкенда** — Render/Railway, чтобы вход работал без localhost.  
7. Дальше по фазам: инфра → прототип IXI → CONNECT/TRADE/REPOSITORIUM.

Всё остальное (блокчейн IXI, кошельки, биржа, Repositorium, обучение, стартапы, медиа, награды, игры) — из этого списка; подключаем по мере готовности фундамента.
